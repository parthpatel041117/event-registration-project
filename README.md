# 🎉 Event Management Web App

A simple and efficient web application to manage events, built using Python and SQLite.  
This project allows users to create, view, and manage events easily.

---

## 🚀 Features

- 📅 Create new events
- 🗂️ Categorize events (Tech, Cultural, etc.)
- 📝 Add event descriptions
- 📊 Store data using SQLite database
- 🔍 View all events in a structured format

---

## 🛠️ Tech Stack

- **Backend:** Python
- **Database:** SQLite
- **Frontend:** HTML, CSS (if used)
- **Version Control:** Git & GitHub
